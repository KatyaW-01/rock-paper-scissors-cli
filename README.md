# Rock Paper Scissors CLI
A Javascript CLI allowing users to play rock paper scissors with the computer
### Features
* Main menu with four options: start game, see stats, reset stats, and quit
* During game, computer chooses a random item
* Display of statistics showing how many games were lost, won, or tied
* Ability to reset statistics back to 0

### Starting the game
  ```bash
  node bin/index.js
  ```
* Select rock, paper, or scissors with the up and down arrows, pressing the enter key once a slection has been made
* A message will display what the computer chose
* Play again, view your stats, reset the stats, or exit
